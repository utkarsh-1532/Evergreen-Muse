# **App Name**: Evergreen Muse

## Core Features:

- User Authentication: Secure user registration, login, and email verification using Firebase Authentication.
- Habit Tracking: Create and track daily/weekly habits with streak tracking. Uses a simple checkbox UI.
- Learning Projects: Track progress on learning projects or skill development using milestones or hours spent.
- Creative Journal: A rich-media journal for capturing thoughts, including title, body text (Rich text or Markdown), and date.
- Media Attachments: Upload images to Firebase Storage with optional captions.
- Soundtrack Association: Associate journal entries with music by pasting a song link (Spotify/YouTube) or typing a song name. Suggest appropriate songs or instrumental music with a tool that considers the text of the entry.
- Journal Feed: Display journal entries in a visually appealing vertical timeline with images, captions, and associated music.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) for primary actions, evoking nature and growth.
- Background color: Off-white/cream (#F8F8FF), providing a soft, minimalist backdrop.
- Accent color: Charcoal gray (#36454F) for text, ensuring readability and a subtle contrast.
- Body and headline font for UI elements: a clean Sans-Serif (like Inter or Geist).
- Body and headline font for journal entries: a high-quality Serif (like Merriweather or Playfair Display) for a 'book-like' readability.
- Use Lucide-React icons throughout the application for a consistent and modern aesthetic.
- Mobile-first, app-like layout with a bottom navigation bar on mobile and a sidebar on desktop.
- Cards with rounded corners and generous spacing to create a clean and breathable interface.
- Subtle animations on page transitions and interactions to enhance user experience.
